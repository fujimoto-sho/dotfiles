# dotfiles

## 手順

dotfilesのクローン

```bash
$ cd ~
$ git clone git@github.com:okutani-t/dotfiles.git
```

権限付与

```bash
$ chmod +x ~/dotfiles/homebrew_install.sh
$ chmod +x ~/dotfiles/install.sh
```

実行

```bash
$ ~/dotfiles/homebrew_install.sh
$ ~/dotfiles/install.sh
```
